# Trading Graphs Dashboard

A clean, modern trading dashboard built with **Next.js + TypeScript** for visualizing stock charts and technical indicators across **US & Indonesian markets**.

---

## Features
- Candlestick & price charts  
- Plug-and-play technical indicators (SMA, MACD, RSI, etc.)  
- Pinned watchlist (max 5 stocks)  
- Multi-market support (NASDAQ, NYSE, IDX)  
- Live data with offline fallback (charts never go blank)

---

## Run Locally
```bash
npm install
npm run dev
